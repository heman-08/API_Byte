This project contains fice modular "Byte Projects" designed to master the modern iOS tech stack (2025-2026). Each prokect focuses on a core technical pillar required by MNCs for enterprise level app development.
PROJECT-01: THE FOREVER NOTE
Core concept - Persistence and Architecture 
Tech Stack - SwiftData, SwiftUI, MVVM
Demonstration - This project moves beyond simple @State variables to use the @Model macro for permanent local storage.

PROJECT-02: THE INSIGHT ENGINE 
Core concept - Modern Networking 
Tech Stack - URL Session, async/await, Codable, JSON
Demonstration- This project shows the ability to fetch data from external REST APIs and use the Codable protocol to safely transform JSON into Swift objects while handling loading and error states professionally.

PROJECT-03: The Siri Sidekick 
Core Concept - System Integration & Apple Intelligence
Tech Stacks - App intents framework, Siri Shortcuts 
Demonstration - This project exposes app features to Siri, Spotlight, and Shortcuts using AppIntent and AppShortcutsProvider.

PROJECT-04: The Permission Gate 
Core - System Data and Privacy 
Tech stacks - HealthKit, HKHealthStore, Privacy PLIST management.
Demonstration - This project implements a secure bridge to HealthKit, managing asynchronous permission requests and ensuring compliance with Apple's strict privacy-first guidelines.

PROJECT-05: The Progress Painter 
core - Date visualization
Tech stacks - Swift cHarts, Bar Mark ,Rule mark
Demonstration - Builds interactive bar graphs and "Goal" lines, proving the developer can create the insightful dashboards found in modern health and fintech applications.
